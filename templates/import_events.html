Upload Events

Please upload a list of observed events in either Microsoft Excel or Comma-Separated Values (.csv) file format. The uploaded spreadsheet must, at minimum, contain the following columns:
  • kingdom
  • phylum
  • class
  • order
  • family
  • genus
  • species
You many have any number of additional columns in the spreadsheet, but the columns above must be present in order for HeritageRoots to import and link events correctly. Note that column names are case-sensitive.
